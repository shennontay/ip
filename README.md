## Setting Up

### Prerequisites
- JDK 17
- IntelliJ IDEA

### Run the program
1. Download the latest `nomnom.jar` from the [Releases](https://github.com/<shennontay>/ip/releases).
2. Open a terminal in the folder containing the jar.
3. Run: java -jar nomnom.jar
